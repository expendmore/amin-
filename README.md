# Sensy AI Enterprise PlatformSensy AI is an enterprise-grade customer communications, conversational AI chatbot, and workflows automation platform.## Features- **Multi-Tenant Separation**: Dynamic workspace scoping and secure database RLS policies.- **Workflow Engine**: Compiles canvas steps and executes node actions or conditional branches.- **Enterprise RAG**: Overlapping sliding text chunker and hybrid semantic searches.- **Multi-Channel Notifications**: SMTP/Resend emails, FCM device push alerts, and real-time in-app alerts.- **Auto-Failover Payments**: Instantly switches gateway checkouts (Stripe/Razorpay) and computes GST.- **Realtime Gateway**: Live user typing indicators and streaming AI text tokens.- **Observability SDK**: JSON logs key masking, metrics tracking, and OpenTelemetry trace correlation.- **DevOps Stacks**: Standalone Docker builds, nginx reverse proxies, and GitHub Actions pipelines.## Getting Started### Local Setup1. Clone the repository and install dependencies:   ```bash   npm install --legacy-peer-deps   ```2. Initialize database client generators:   ```bash   npx prisma generate   ```3. Boot the local development stack:   ```bash   npm run dev   ```### Execution of TestsTo verify all 44 Vitest test assertions:```bashnpm run test```### Production Build compilationTo build Next.js optimized bundles:```bashnpm run build```
